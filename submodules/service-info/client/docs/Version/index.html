    Version¶

    Properties¶

    Name Type Description Notes
    BuildVersion Pointer to string The build version and metadata of this release (e.g. prerelease.1; v0.1.2-prerelease.1 == x.y.z-build.metadata). [optional]
    ContainerImages Pointer to []ContainerImage Container images associated with this version. [optional]
    CreatedAt Pointer to time.Time Object creation time in RFC 3339 format. [optional]
    Id Pointer to string Version in semantic versioning format. [optional]
    Stable Pointer to bool Whether this is a stable version. [optional]
    UpdatedAt Pointer to time.Time Object last updated time in RFC 3339 format. [optional]
    XVersion Pointer to int64 The X version of this release (e.g. 0; v0.1.2-prerelease.1 == x.y.z-build.metadata). [optional]
    YVersion Pointer to int64 The Y version of this release (e.g. 1; v0.1.2-prerelease.1 == x.y.z-build.metadata). [optional]
    ZVersion Pointer to int64 The Z version of this release (e.g. 2; v0.1.2-prerelease.1 == x.y.z-build.metadata). [optional]
